Galeria de arte

Este proyecto requirió que construyera una galeria de arte totalmente receptiva a los diseños proporcionados. Usé HTML5, junto con CSS flexbox y vanilla JavaScript para las áreas que requerían interactividad, como el control deslizante de obras de arte. Tambien use por primera vez la barra de progreso de HTML5. El sitio esta hecho con la ariquitectura SPA(single page application), fue hecho de esta manera por que queria entender como funcionaba react pero el sitio esta hecho 100% vanilla js.
