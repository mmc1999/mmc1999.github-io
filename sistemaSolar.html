Sistema solar

En este proyecto use solamente css flexbox. Y los datos los "pinte" con javascript de una API. Me gusto mucho este desafio por que aprendi a visualizar datos depependiendo el archivo en que se encontraba y ademas aprendi a cambiar pseudo-clases con JavaScript(fijarse el color de los enlaces cuando paso por encima el mouse o el color de los botones cuando hago click).

Antecedentes del proyecto

Este proyecto fue un desafío de front-end de Frontend Mentor. Es una plataforma que le permite practicar la construcción de sitios web con un diseño y un resumen del proyecto. Cada desafío incluye diseños móviles y de escritorio para mostrar cómo debe verse el sitio web en diferentes tamaños de pantalla. La creación de estos proyectos me ha ayudado a refinar mi flujo de trabajo y a resolver problemas de codificación del mundo real. Aprendí algo nuevo con cada proyecto, lo que me ayudó a mejorar y adaptar mi estilo.
